React - Experiments

Comparing React against other JavaScript frameworks, like Backbone/Marionette and AngularJS.

  • hello-world - basic, single component use. Makes use of `props`.
  • react-tutorial - the React tutorial - a comments list.
  • react-table - Render a big table with dynamic features from a mock JSON file, using React.
  • marionette-table - Render a big table with dynamic features, using Backbone + Marionette.
  • angular-table - Render a big table with dynamic features, using AngularJS.
  • react-clicker - Very basic component example in react.
